# BRAHMO AI Assistant

Organizational Healthcare Knowledge System powered by AI.

## Overview

BRAHMO is a Knowledge Elicitation and Context Injection platform that retrieves hospital, department, and patient-specific knowledge before generating AI recommendations.

The system ensures healthcare professionals receive context-aware responses instead of generic AI answers.

---

## Features

- Hospital Knowledge Layer
- Department Knowledge Layer
- Patient Knowledge Layer
- AI Recommendation Engine
- Context Injection
- Knowledge Filtering
- Supabase Integration
- Gemini Integration
- Next.js Frontend
- Express Backend
